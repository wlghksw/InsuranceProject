# 회원관리

## 로그인 , 회원가입
회원가입을 통해 사용자 정보를 입력받고 로그인을 하여 페이지에 접근 가능하도록 함. 가입된 계정을 아이디찾기, 비밀번호찾기로 찾을 수 있음 
(카카오톡 API를 추가하여 카카오톡으로 로그인도 가능함)

## 핵심기능
  * **회원가입**
    * 사용자로부터 아이디, 비밀번호, 닉네임, 실명, 전화번호, 성별을 입력받고 회원가입이 완료된 계정으로만 로그인을 시도했을 때 정상 처리되도록 함
  * **로그인**
    * 회원가입이 완료된 계정만 로그인이 가능함. (단, 관리자페이지에서 비활성화 처리된 계정은 로그인이 불가능하다)
  * **아이디찾기**
  * 회원가입을 할 때 입력했던 실명, 전화번호를 입력받아 일치한 계 경우

# 관리자 페이지
로그인 페이지에서
```bash
# 등록된 관리자 계정
id : admim
pw : 1234
```
다음 계정으로 로그인 했을 때 관리자 페이지로 이동

* **핵심 기능**
  * 역할 관리
    * 유저 API로 사용자가 로그인 상태인지 파악하고 계정에 등록되어 있는 역할을 확인하고 표시한다
    * ADMIN은 USER에게 ADMIN 권한 부여가 가능하다
   (이때 역할은 ADMIN, USER로 구분)
  * 로그인 가능/비가능 상태 설정
    * 활성화, 비활성화로 나누어서 회원가입이 되어 있는 계정이지만 비활성화로 설정할 경우 해당 계정의 로그인은 불가능한 상태가 됨
  * 보험 상품 관리
    *기존 보험 상품 목록에 새로운 보험 상품을 ADMIN이 추가할 수 있음

#  보험료 계산기
사용자로부터 **정보**를 입력받았을 때 예상 보험료를 제공하는 페이지

* **핵심 기능**

* 더미데이터를 사용. 보험 별로 가상 계산 공식에 맞춰 보험료를 예측

  * 보험별 계산 과정
  * 1. 암보험
    * 사용자로부터 나이와 보험료를 입력 받음
    * 기본 보험료를 10000원으로 설정
    * 입력받은 나이가 40세를 초과할 경우 2.1배, 아닐 경우 1.5배의 나이요율을 적용
    * 입력받은 보장금액에서 1000원 당 4000원의 보험료 책정
    * 최종적으로 월 예상 보험료를 계산하여 책정 금액을 표시
      ```bash
      # 암보험 월 예상 보험료 계산 공식
      월 예상 보험료 = [10000(기본보험료) * AgeRate(나이요율)] + [coverage(보장금액)/ 1000 * 4000]
      ```
  * 2. 실손의료보험
    * 사용자로부터 나이와 성별을 입력 받음
    * 기본 보험료를 15000원으로 설정
    * 입력받은 나이가 40세를 초과하는 경우 1.5배, 아닐경우 1.1배의 나이요율을 적용
    * 입력받은 성별이 남성일 경우 1.2배, 여성일 경우 1.0배의 성별 요율을 적용
    * 최종적으로 월 예상 보험료를 계산하여 책정 금액을 표시
      ```bash
      # 실손의료보험 월 예상 보험료 계산 공식
      월 예상 보험료 = 15000(기본보험료) * AgeRate(나이요율) * genderRate(성별요율)
      ```
  * 3. 보장성보험
    * 사용자로부터 나이와 사멍보험금을 입력받음
    * 기본 보험료를 20000으로 설정
    * 입력받은 나이가 40세를 초과하는 경우 1.8배, 아닐경우 1.2배의 나이요율을 적용
    * 사망보험금은 1억당 8000원으로 환산
      ```bash
      이익비용 계산 공식
      BenefitCost(이익비용) = (사망보험금 / 10000) * 8000
      
      # 보장성보험 월 예상 보험료 계산 공식
      월 예상 보험료 = [20000(기본보험료) * AgeRate(나이요율)] + BenefitCost(이익비용)
      ```
    * 4. 자동차보험
      * 사용자로부터 나이와 운전경력년도를 입력받음
      * 기본보험료를 70만원으로 설정
      *  입력받은 나이가 26세 미만일 경우 1.5배, 나이가 26세 이상이고 30세 미만일 경우 1.2배, 그 외의 나이는 1.0배의 나이요율을 적용
      *  운전경력년도가 3년 미만일 경우 1.3배, 3년 이상일 경우 1.0배 운전경력년도에 따른 요율을 적용
      *  (*단, 차량 종류, 사고이력, 특약, 주행거리, 위반 등에 따른 보험료는 계산에서 제외함.) => 향후 계획에 추가
      ```bash
      # 자동차보험 월 예상 보험료 계산 공식
      월 예상 보험료 = 700000(기본보험료) * AgeRate(나이요율) * expRate(운전경력년도 요율)
      ```
