# 보험 용어 전문 AI 챗봇 

**Sentence-BERT**와 **Faiss**를 기반으로, 사용자가 질문한 보험 용어의 의미를 정확하게 찾아주고 관련 용어까지 추천해주는 AI 챗봇 서비스입니다.

-----

## 핵심 특징

  * **의미 기반의 정확한 답변**: 사용자가 입력한 질문의 핵심 의도를 파악하는 **시맨틱 검색(Semantic Search)** 기술을 사용합니다. 단순히 키워드가 일치하는 것을 넘어, 문맥적으로 가장 유사한 보험 용어를 찾아 정확한 답변을 제공합니다.
  * **고성능 벡터 검색**: Facebook AI의 **Faiss** 라이브러리를 활용하여, 수많은 보험 용어 벡터 중에서 가장 유사한 답변을 매우 빠른 속도로 찾아냅니다.
  * **관련 용어 추천**: 사용자가 질문한 용어와 의미적으로 유사한 \*\*'함께 찾아본 용어'\*\*를 추가로 추천하여 사용자의 연관 학습을 돕습니다.
  * **간편한 웹 인터페이스**: **Flask**로 구축된 간단하고 직관적인 웹 화면을 통해 누구나 쉽게 질문하고 답변을 확인할 수 있습니다.

-----

## 작동 원리

1.  **데이터 준비**: `보험용어정리_new.csv`에 있는 보험 용어와 그 의미를 읽어옵니다.
2.  **임베딩 생성**: **Sentence-BERT** 모델(`jhgan/ko-sbert-nli`)을 사용하여 각 용어의 '의미'를 나타내는 고차원 벡터(임베딩)로 변환합니다.
3.  **Faiss 인덱싱**: 변환된 모든 용어 벡터를 **Faiss** 인덱스에 저장하여, 유사도 검색을 위한 초고속 데이터 구조를 구축합니다.
4.  **사용자 질문 처리**:
      * 사용자의 질문이 들어오면, 이 질문 역시 Sentence-BERT 모델을 통해 벡터로 변환됩니다.
      * Faiss 인덱스에서 사용자의 질문 벡터와 가장 가까운(유사도가 높은) 상위 K개의 보험 용어 벡터를 찾아냅니다.
      * 가장 유사도가 높은 용어를 **정확한 답변**으로, 나머지 용어들을 \*\*'함께 찾아본 용어'\*\*로 사용자에게 보여줍니다.

## 설치 및 실행

모든 명령어는 **`python_chatbot`** 폴더 안에서 실행합니다.

### 1단계: 폴더 이동 및 의존성 설치

```bash
# 1. 챗봇 폴더로 이동합니다.
cd python_chatbot

# 2. 의존성 목록 파일을 이용해 필요한 라이브러리를 설치합니다.
pip install -r requirements.txt
```

### 2단계: 모델 학습 (최초 1회만 실행)

AI 모델이 답변을 생성할 수 있도록 학습을 진행합니다.

```bash
# (현재 위치: python_chatbot 폴더)
python train_model.py
```

  * 실행이 완료되면 `insurance_terms.pkl`과 `faiss_index.bin` 파일이 생성됩니다.

### 3단계: 챗봇 서버 실행

Flask 웹 서버를 시작합니다.

```bash
# (현재 위치: python_chatbot 폴더)
python app.py
```

  * 서버가 정상적으로 실행되면 **`http://127.0.0.1:5001`** 주소로 접속하여 챗봇을 사용할 수 있습니다.
